This Project is modified from
	https://github.com/chunikuo/stm32F4_GPIO_Demo
which clean the unnecessary files from stm32F4_GPIO_Demo.

To compile this project
$ make

To write program onto the stm32f429i discovery board
$ make flash

To clean the project
$ make clean
